# Moléculas

Moléculas são componentes de design mais complexos que átomos. Elas são formadas pela combinação de átomos para criar elementos mais funcionais e com maior significado na interface.

## O que são Moléculas no Design System?

Em nossa biblioteca de componentes, as moléculas representam elementos compostos que:

- Combinam múltiplos componentes atômicos
- Possuem funcionalidades específicas e bem definidas
- Podem ser reutilizadas em diferentes contextos
- Mantêm consistência visual e comportamental

## Quando usar Moléculas?

Moléculas são ideais quando você precisa:
- Criar componentes interativos mais complexos
- Combinar múltiplos elementos em uma única unidade funcional
- Estabelecer padrões reutilizáveis de interface
- Manter consistência em diferentes partes da aplicação
