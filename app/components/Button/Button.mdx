# Botão

O componente Botão é um elemento fundamental de interface que permite interação do usuário através de cliques ou toques.

## Visão Geral

Este botão foi desenvolvido seguindo as diretrizes do nosso Design System, oferecendo:
- Design moderno e consistente
- Feedback visual ao interagir
- Acessibilidade incorporada
- Personalização flexível através de props

## Características

- **Estilização Base**: Fundo azul (#2D5BFF) com texto branco
- **Bordas**: Arredondamento de 8px
- **Espaçamento**: Padding horizontal de 32px e vertical de 12px
- **Interatividade**: Efeito hover com transição suave
- **Acessibilidade**: Suporte completo a navegação por teclado
